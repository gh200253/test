# شجرة العائلة التفاعلية

تطبيق ويب تفاعلي لإنشاء ورسم شجرة العائلة باستخدام D3.js

## المميزات

- إضافة الأعضاء بطريقة ذكية (سلسلة الأسماء)
- إضافة يدوية للأعضاء
- تصميم تفاعلي مع إمكانية التكبير والتصغير
- حفظ البيانات محلياً في المتصفح
- واجهة باللغة العربية

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript 
- D3.js  

## كيفية الاستخدام

1. افتح الملف `final.html` في المتصفح
2. استخدم مربع "إضافة ذكية" لإضافة سلسلة من الأسماء
3. أو استخدم "إضافة يدوية" بعد تحديد عضو من الشجرة
